Vagrants
======================

This is a collection of vagrantfiles for various tasks

Rebuildatron
-------------------

This sets up a pxeboot build server to help rebuild from scratch

chef-base-dev
-------------------

This sets up the machines to test out my chef-base and chef-master cookbooks to
simulate my NAS.
